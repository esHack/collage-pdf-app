# Collage PDF App

A modern web application for creating PDF documents from images with drag-and-drop functionality, rotation, resizing, and text support.

## Features

- Upload multiple images (including HEIC format)
- Drag, resize, and rotate images
- Add and edit text
- Multi-page support
- Export to A4 PDF format
- Modern gradient UI

## Tech Stack

- **Frontend**: React 18, Lucide React icons
- **Backend**: Node.js, Express, Puppeteer
- **Deployment**: Vercel

## Local Development

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```

## Deployment

This app is configured for Vercel deployment:

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically
